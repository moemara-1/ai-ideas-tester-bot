# Outreach Strategy & Lead-Gen Workflow

## 1. Target ICP (Ideal Customer Profile) Assumptions
- **Industry:** Local service businesses (e.g., Plumbers, Roofers, Electricians, HVAC, Landscapers) and traditional B2B professional services (e.g., Accountants, Legal firms) with outdated, slow, or absent web presences.
- **Criteria:**
  - Has a Google Maps presence but either no website linked or an outdated one.
  - Can identify an active contact email or contact form.
  - High customer lifetime value (LTV), meaning a modern, lead-capturing website provides immediate, measurable ROI.

## 2. Channel Priorities
- **Primary Channel:** Cold Email via Resend. This is highly scalable, automatable, and perfectly matches our pipeline capabilities.
- **Why Email:** It allows us to seamlessly send a personalized link to the customized demo site generated by the Builder Agent. The demo site acts as the ultimate low-friction "hook".
- **Secondary Channels:** Excluded for MVP (LinkedIn, SMS, direct mail are out of scope as per the execution brief).

## 3. Messaging Angles
- **Angle 1 (The "Show, Don't Tell" Value First):** "Hey [Name], we noticed your Google Maps listing didn't have a modern website linked. We actually went ahead and built a custom demo site for [Business Name]. Click here to view it: [Cloudflare URL]. If you like it, we can make it live today for a flat [Price/Subscription]."
- **Angle 2 (The Competitive Upgrade):** "Hey [Name], your competitors are capturing local leads online with modern sites. We built a high-converting landing page tailored specifically for [Business Name] to help capture those same leads. Check out the working prototype here: [Cloudflare URL]."
- **Tone:** Direct, value-first, zero-friction. The customized demo site does the heavy lifting and selling.

## 4. First-Pass Lead-Gen + Outreach Workflow
1. **Scouting (Scout Agent):** Query Google Maps Places API for structured queries like "HVAC contractors in [City], [State]". Filter for businesses with missing website links or poor review counts indicating a lack of digital optimization.
2. **Intel (Intel Agent):** Scrape the business's web footprint (if any) or Google Places data to extract contact emails and a summary of their specific services. This summary is crucial context for the site generation.
3. **Building (Builder Agent):** Use the gathered intel to prompt an OpenRouter free model to generate a tailored, single-page Tailwind HTML landing page. Deploy this page via Cloudflare Pages and record the unique live URL.
4. **Outreach (Outreach Agent):** Draft the initial cold email using a Resend template, dynamically inserting the business's details and the Cloudflare URL. Send the email with a clear Call-To-Action (e.g., "Reply if you want to keep this site").
5. **Closing & Scheduling (Closer/Scheduler Agents):** Listen to Resend webhooks for replies. The Closer Agent analyzes intent—if positive, it automatically replies with a Stripe payment link. If a meeting is requested, the Scheduler Agent replies with a calendar link.
